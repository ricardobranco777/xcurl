# xcurl
xcurl gets the HTML from dynamic webpages where curl fails

## Usage:

```
usage: xcurl [-h] [-t TIMEOUT] url

positional arguments:
  url

optional arguments:
  -h, --help            show this help message and exit
  -t TIMEOUT, --timeout TIMEOUT
                        Timeout in seconds
```

## Requirements

- Python 3
- Beautiful Soup
- Python Selenium
